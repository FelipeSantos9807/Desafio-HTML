Estrutura Básica do HTML

A estrutura básica do HTML consiste de 3 tags, a inicial HTML dentro dela vem HEAD e BODY.

A tag HEAD como sua tradução sujere é a "cabeça" da pagina onde você pode definir coisas como o titulo na tag Title que será exibido na guia de navegação.

A tag BODY é o "corpo" da página onde ficam textos, imagens, interações com o usuário.

Falando sobre Tags

Agora mais algumas Tags do HTML e suas funções:

  • Tag: <'i> é utilizada para deixar palavras, textos em itálico<'/i>;
  • Tag: <'strong> é utilizada para deixar palavras, textos em negrito<'/strong>, pode ser utilizada em conjunto com a tag <'i>;
  • Tag: <'mark>é utilizada para marca trechos de textos<'/mark>;
  • Tag: <'u> é utilizada para sublinhar uma palavra ou trecho do texto<'/u>';
  • Tag: <'/hr> é utilizada para traçar uma linha divisoria hotizontal.Essa tag já é fechada no inicio, pois não possui tag de fechamento;
  • Tag: <'small> é utilizado para fazer textos com uma fonte menor <'/small>;
  • Tag: <'sub> é utilizada para textos subscritos, um exemplo de uso seriam fórmulas químicas<'/sub>;
  • Tag: <'sup> é utilizada para textos sobrescritos, um exemplo de uso é a representação de ponteciação <'/sup>;
  • Tag: <'blockquote> é ultilizado para trechos de referencia dentro do texto exemplo:
    "A <'blockquote>tag especifica uma seção citada de outra fonte."

Atributos das Tags

Alguns atributos que as tags podem ter:

  • ID=""- Atributo para identificação;
  • Style=""-Atributo que contém informações de estilo para um documento ou uma parte do documento. As informações de estilo específico estão contidas dentro deste elemento, geralmente no CSS;
  • Class=""-Atributo voltado para o CSS usado para padronizar configurações da classe criada;

    • Atributos com usos expecificos:

  • type="number"-Atributo que só permite números no campo de texto;
  • src=""-Atributo onde é colocado o Endereço de imagem;
  • width=""-Atributo para definir largura da imagem.

Textos

Tags para Textos:

  • TAG: <'h1~h6> São divididos em níveis de titulo, subtitulo e assim em diande;
  • TAG: <'p> É usada para definir um paragráfo, quebra de linha;

Listas Ordenadas e Não Ordenadas

Nesse ponto vamos ver os 2 tipos de listas e suas funções

Lista Ordenada

Uma lista ordenada tem uma estrutura númerica crescente com o número de itens registrado.

Para criar uma lista Ordenada em HTML usamos a TAG <'OL> e a TAG <'li> para os itens.

  1. Primeiro item
  2. Segundo item
  3. Terceito item

Lista Não Ordenada

Diferente da Lista Ordenada essa lista não ordena os itens com números, mas representa os itens com um símbolo.

Para criar uma lista Não Ordenada em HTML usamos a TAG <'UL> e a TAG <'li> para os itens.

  • Primeiro item
  • Segundo item
  • Terceito item

O HTML também permite que façam uma lista dentro da outra, e mesclar os dois tipos também é possivel.

  1. Primeiro item
  2. Segundo item
    • Primeiro item
    • Segundo item
    • Terceito item
  3. Terceito item

Links

Criando Links

Para criar links usamos a TAG: <'a> com o atributo href="" onde colocamos o endereço do site que queremos fazer a ligação:

O atributo target="_blank" faz com que o link seja aberto em outra Aba do navegador

O atributo title="Cria um pequeno texo de ao passar o mouse pelo link"

Exemplo: <'a href="www.Google.com" title="Para pesquisar clicle aqui" target="_blank">Google<'/a> a parte grifada é a que será mostrada como link para o usuário.

Também é possivel criar links que vão ir para locais dentro da propria página, desde que o local tenha um id.

Exemplo:<'h3 id="T01">Titulo 01<'/h3>, Para criar um link para esse local da pagina fariamos assim:

<'a href="#T01">Ir para Titilo 01<'/a>
